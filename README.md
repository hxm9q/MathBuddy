# MathBuddy

iOS-приложение для тренировки и практики математических навыков. Позволяет решать задачи, отслеживать прогресс, изучать справочные материалы и улучшать свои математические способности через интерактивный и адаптивный интерфейс.

## Основные требования и особенности проекта

**Архитектура:** SwiftUI + MVVM с чистым разделением слоёв  
**Сервисы:** MathCalculationService, MathFormulaService, ProgressService  
**Состояние:** ViewState + BaseViewModel для управления экраном  
**Утилиты:** Logger для отладки и отслеживания событий  

**Основные экраны приложения:**
1. Practice — решение математических задач
2. Progress — отслеживание прогресса пользователя
3. Overview — обзор текущих результатов, рандомная формула дня
4. Reference — справочные материалы по математике
5. Settings — управление настройками приложения, сброс прогресса

## Описание приложения (для App Store Connect)

**Название:** MathBuddy  
**Подзаголовок:** Учись и тренируй математику каждый день  

**Описание:**  
MathBuddy — удобное iOS-приложение для практики и изучения математики. Решайте задачи, отслеживайте свой прогресс, изучайте формулы и справочные материалы. Приложение создано с упором на простоту использования и эффективность обучения.
